# Ticket Sales Analysis and Dashboard

**Project Goal:**  
Analyze ticket sales, identify key metrics, and build a dashboard to support business decisions.

**Project Description:**  
- Conducted exploratory data analysis (EDA) — structure, missing values, distributions  
- Calculated main sales metrics and performance indicators  
- Built an interactive dashboard to visualize key metrics  
- Performed segmentation and identified main sales patterns

**Skills and Technologies:**  
- Python (pandas, numpy, matplotlib, seaborn)  
- Data analysis and visualization  
- Dashboard creation (Yandex DataLens)  
- Business reporting and insights

**Repository Structure:**
```

├── README.md # Russian version
├── README_en.md # English version
├── ticket_sales_analysis.ipynb # analysis notebook with charts

```
