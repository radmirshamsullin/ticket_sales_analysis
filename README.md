# Анализ продаж билетов и дашборд

**Цель проекта:**  
Провести анализ продаж билетов, выявить ключевые метрики и построить дашборд для поддержки бизнес-решений.

**Описание проекта:**  
- Проведен первичный анализ данных (EDA) — структура, пропуски, распределения  
- Построены основные метрики продаж и показатели эффективности  
- Создан интерактивный дашборд для визуализации ключевых показателей  
- Проведена сегментация и выявлены основные закономерности продаж

**Навыки и технологии:**  
- Python (pandas, numpy, matplotlib, seaborn)  
- Анализ данных и визуализация  
- Работа с дашбордами (Yandex DataLens)  
- Подготовка отчётов и выводов для бизнеса

**Структура репозитория:**
```
ticket_sales_analysis/
├── README.md                   # текущий файл
├── README_en.md                # англоязычная версия

```
